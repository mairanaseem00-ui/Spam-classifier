# Spam Classifier
A beginner machine learning project that classifies spam and ham messages using Natural Language Processing (NLP) techniques with Python and scikit-learn.

## Features
- Loads and processes a spam message dataset
- Converts text into numerical data using CountVectorizer
- Uses a Naive Bayes classifier for prediction
- Splits data into training and testing sets
- Calculates model accuracy
- Predicts custom spam messages

## Technologies Used
- Python
- pandas
- scikit-learn

## Machine Learning Concepts
- Natural Language Processing (NLP)
- Text Classification
- Bag of Words
- Naive Bayes Classification

The model achieves around 95–99% accuracy depending on the dataset split.
